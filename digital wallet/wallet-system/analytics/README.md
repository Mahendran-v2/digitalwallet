# Analytics
Analytics Layer
Use Power BI Desktop with the PostgreSQL connector.
Start PostgreSQL and run database/schema.sql.
Connect Power BI to the same database used by the FastAPI app.
Use analytics/powerbi_queries.sql for KPI cards and charts.
Suggested visuals:
Total Users
Total Transactions
Total Money Transferred
Fraud Alerts
Active Users
Transaction Trend
Top Users
Risk Score Distribution
Daily Transactions

Power BI queries and analytics helpers for the wallet system.
